## 🔧 Требуется от пользователя

**КРИТИЧНО:** Необходимо указать правильный пароль для PostgreSQL.

Текущие настройки в `.env`:
```
DATABASE_URL=postgresql://postgres@localhost:5432/accounting_db
DB_USER=postgres  
DB_PASSWORD=
```

**Варианты решения:**
1. Указать пароль пользователя `postgres` в PostgreSQL
2. Создать нового пользователя с паролем
3. Использовать доверительную аутентификацию (trust) в pg_hba.conf
