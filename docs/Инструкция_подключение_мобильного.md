# 📱 Инструкция: Подключение мобильного приложения SmartPOS

## Для клиентов без отдельного сервера (через WiFi)

---

## 🌐 Как это работает

Сервер встроен прямо в десктопное приложение SmartPOS Pro.
Когда вы запускаете программу на компьютере — сервер запускается автоматически.
Мобильные устройства подключаются к нему через вашу WiFi сеть.

```
┌─────────────────────────────────────────────────┐
│                  WiFi Роутер                    │
│              (ваша локальная сеть)              │
└────────┬────────────────────────┬───────────────┘
         │                        │
         ▼                        ▼
┌─────────────────┐      ┌─────────────────┐
│   🖥️ Компьютер  │      │  📱 Телефон     │
│                 │      │                 │
│  SmartPOS Pro   │      │  SmartPOS       │
│  (десктоп)      │◄────►│  (мобильное)    │
│                 │      │                 │
│  Встроенный     │      │  Подключается   │
│  сервер         │      │  по WiFi        │
│  порт: 5000     │      │                 │
└─────────────────┘      └─────────────────┘
```

---

## 📋 Требования

| Что нужно | Описание |
|-----------|----------|
| 🖥️ Компьютер | С установленным SmartPOS Pro |
| 📱 Телефон | С установленным мобильным приложением SmartPOS |
| 📶 WiFi | Оба устройства в **одной** WiFi сети |
| 💡 Компьютер включён | Сервер работает пока компьютер включён |

## 🔧 Пошаговая настройка

### Шаг 1: Запустите SmartPOS Pro на компьютере

Откройте приложение **SmartPOS Pro** на вашем компьютере.
Сервер запустится автоматически в фоновом режиме.

```
[Запуск SmartPOS Pro] → [Сервер стартует автоматически] → [Готов к подключениям]
```

---

### Шаг 3: Подключите телефон к WiFi

Убедитесь что ваш **телефон** подключён к **той же WiFi сети**,
что и **компьютер** с SmartPOS Pro.

> ⚠️ **Важно!** Телефон и компьютер ОБЯЗАТЕЛЬНО должны быть
> в одной WiFi сети. Если они в разных сетях — подключение не сработает.

---

### Шаг 4: Настройте мобильное приложение

1. Откройте **SmartPOS** на телефоне
2. Перейдите в **Настройки** (⚙️)
3. Найдите поле **"Адрес сервера"**
4. Введите адрес, который показан на экране компьютера

```
┌─────────────────────────────┐
│ ⚙️ Настройки                │
├─────────────────────────────┤
│                             │
│ Адрес сервера:              │
│ ┌─────────────────────────┐ │
│ │ http://192.168.1.45:5000│ │
│ └─────────────────────────┘ │
│                             │
│ ┌─────────────────────────┐ │
│ │     💾 Сохранить        │ │
│ └─────────────────────────┘ │
│                             │
│ ✅ Подключено к серверу     │
│                             │
└─────────────────────────────┘
```

Или **сканируйте QR-код** с экрана компьютера.

---

### Шаг 5: Войдите в систему

После подключения к серверу:

1. Введите **логин** и **пароль** кассира
2. Нажмите **"Войти"**
3. Приложение готово к работе! 🎉

## ❓ Частые вопросы

### «Не удаётся подключиться к серверу»

| Проверьте | Решение |
|-----------|---------|
| WiFi | Телефон и ПК в одной сети? |
| Компьютер | SmartPOS Pro запущена? |
| Адрес | Правильно введён IP и порт? |
| Антивирус | Не блокирует порт 5000? |
| Брандмауэр | Разрешён доступ для Node.js? |

### «Сервер не запускается»

Откройте **Настройки → Сервер и мобильные** и нажмите
**"Перезапустить сервер"**.

### «IP-адрес изменился»

IP может меняться при перезагрузке роутера.
Просто откройте вкладку **"Сервер и мобильные"** и
введите новый адрес на телефоне.

### «Можно ли подключить несколько телефонов?»

Да! К одному серверу можно подключить **неограниченное**
количество мобильных устройств в одной WiFi сети.
