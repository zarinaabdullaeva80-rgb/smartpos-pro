запусти # Сборка .exe файлов для 1С Бухгалтерия и Админ панели

Этот документ содержит инструкции по сборке desktop приложений (.exe) для Windows.

## Требования

- Node.js >= 18.0.0
- npm или yarn
- Windows (для сборки .exe файлов)

## 1. Сборка 1С Бухгалтерия

### Установка зависимостей

```bash
cd client-accounting
npm install
```

### Разработка (с Electron)

```bash
# Запустить в режиме разработки с Electron
npm run electron:dev
```

### Сборка .exe файла

```bash
# Собрать production версию
npm run electron:build
```

Готовый .exe файл будет находиться в:
```
client-accounting/dist-electron/
```

Файл установщика: `1С Бухгалтерия Setup X.X.X.exe`

## 2. Сборка Админ панели

### Установка зависимостей

### Разработка (с Electron)

```bash
# Запустить в режиме разработки с Electron
npm run electron:dev
```

### Сборка .exe файла

```bash
# Собрать production версию
```

## Установка приложений

1. Запустите файл `Setup.exe`
2. Следуйте инструкциям установщика
3. Приложение будет установлено в `C:\Program Files\`
4. Ярлыки будут созданы на рабочем столе и в меню Пуск

## Настройка подключения к серверу

После установки приложений, убедитесь что:

1. Backend сервер запущен на доступном адресе
2. В файлах конфигурации указан правильный URL сервера:
   - Для 1С: `client-accounting/src/services/api.js`
   - Для Админ: `client-admin/src/App.jsx`

По умолчанию используется `http://localhost:5000/api`

## Иконки приложений

Для установки собственных иконок:

1. Создайте файл `icon.ico` (256x256 px)
2. Поместите его в папки:
   - `client-accounting/build/icon.ico`
   - `client-admin/build/icon.ico`
3. Пересоберите приложения

## Обновление приложений

Для выпуска новой версии:

1. Обновите версию в `package.json`
2. Выполните сборку заново
3. Распространите новый установщик пользователям

## Отладка

Если приложение не запускается:

1. Проверьте логи в `%APPDATA%\1С Бухгалтерия\logs`
2. Убедитесь что backend сервер доступен
3. Проверьте настройки файервола Windows

## Размер приложений

Примерный размер установщиков:
- 1С Бухгалтерия: ~80-100 MB
- Админ панель: ~70-90 MB

Размер включает:
- Chromium engine (Electron)
- Node.js runtime
- React приложение
- Зависимости

## Поддержка других ОС

Для сборки под другие операционные системы:

### macOS
Добавьте в `package.json` → `build`:
```json
"mac": {
  "target": "dmg",
  "icon": "build/icon.icns"
}
```

### Linux
Добавьте в `package.json` → `build`:
```json
"linux": {
  "target": "AppImage",
  "icon": "build/icon.png"
}
```

## Конфигурация electron-builder

Полная конфигурация находится в `package.json` → `build`.

Основные параметры:
- `appId` - уникальный ID приложения
- `productName` - отображаемое имя
- `directories.output` - папка для сборки
- `win.target` - формат установщика (nsis)
- `nsis` - настройки установщика

## Автоматическая сборка

Для CI/CD можно использовать:

```bash
npm run electron:build -- --publish never
```

Это соберет приложение без публикации в репозиторий обновлений.
