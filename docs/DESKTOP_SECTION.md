## Сборка Desktop приложений (.exe)

Оба приложения (1С Бухгалтерия и Админ панель) могут быть собраны как отдельные desktop приложения для Windows.

### Быстрый старт

**1С Бухгалтерия:**
```bash
cd client-accounting
npm install
npm run electron:build
```

**Админ панель:**
```bash
cd client-admin
npm install
npm run electron:build
```

Готовые установщики будут в папках `dist-electron/`

### Особенности desktop версий

✅ Автономные окна приложений  
✅ Нет необходимости в браузере  
✅ Отдельные ярлыки на рабочем столе  
✅ Установка через NSIS installer  
✅ Автообновления (опционально)  

### Системные требования

- Windows 7 и выше
- 100+ MB свободного места
- Backend сервер должен быть доступен по сети
