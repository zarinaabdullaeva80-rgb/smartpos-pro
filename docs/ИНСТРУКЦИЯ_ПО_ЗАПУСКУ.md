# Инструкция по запуску 1С Бухгалтерия

## ✅ EXE файл успешно создан!

### Расположение

Готовое приложение находится в папке:
```
client-accounting\dist-electron\win-unpacked\
```

### Файлы приложения

- `1С Бухгалтерия.exe` - главный исполняемый файл (168.62 MB)
- `ЗАПУСК.bat` - быстрый запуск приложения
- `resources\app.asar` - упакованное React-приложение (117 MB)
- Дополнительные библиотеки Electron и Chromium

### Как запустить

#### Вариант 1: Двойной клик по EXE файлу
1. Откройте папку `client-accounting\dist-electron\win-unpacked\`
2. Дважды кликните на файл `1С Бухгалтерия.exe`

#### Вариант 2: Через bat файл
1. Откройте папку `client-accounting\dist-electron\win-unpacked\`
2. Дважды кликните на файл `ЗАПУСК.bat`

### Требования для работы

1. **Запущенный backend сервер**
   - Убедитесь что сервер запущен на `http://localhost:5000`
   - Для запуска сервера: 
     ```bash
     cd server
     npm run dev
     ```

2. **База данных PostgreSQL**
   - Сервер должен иметь доступ к БД
   - Убедитесь что PostgreSQL запущен

3. **Операционная система**
   - Windows 10 или выше
   - 64-битная версия

### Распространение приложения

Для переноса на другой компьютер:

1. Скопируйте всю папку `win-unpacked` на целевой компьютер
2. Убедитесь что backend сервер доступен по сети
3. Запустите `1С Бухгалтерия.exe`

### Настройка подключения к серверу

По умолчанию приложение подключается к `http://localhost:5000/api`

Для изменения адреса сервера:
1. Откройте файл `src\services\api.js` в исходниках
2. Измените `baseURL` на нужный адрес
3. Пересоберите приложение командой `npm run electron:build`

### Создание установщика (NSIS)

Примечание: При сборке с установщиком возникла ошибка с подписыванием кода.
Текущая версия - portable (не требует установки).

Если необходим установщик с автоматической установкой:
1. Получите сертификат для подписи кода
2. Измените в `package.json`: `"target": "nsis"`
3. Добавьте переменные окружения с сертификатом
4. Запустите `npm run electron:build`

### Troubleshooting

#### Приложение не запускается
- Проверьте что нет запущенных копий приложения
- Проверьте антивирус (может блокировать неподписанное приложение)
- Запустите как администратор

#### Ошибка подключения к серверу
- Убедитесь что backend сервер запущен
- Проверьте firewall
- Проверьте адрес в настройках API

#### Белый экран при запуске
- Backend сервер недоступен
- Проверьте консоль разработчика (F12)
- Проверьте логи сервера
